# SEO Machine

SEO Machine is the TheHues content workspace for US-market English SEO research, planning, writing, and optimization.

## What This Repository Supports

- blog and editorial SEO content
- collection-supporting content strategy
- competitive research and content-gap analysis
- drafting, rewriting, and optimization workflows

## Core Context Files

The main project context lives in:
- `context/brand-voice.md`
- `context/writing-examples.md`
- `context/target-keywords.md`
- `context/internal-links-map.md`
- `context/features.md`
- `context/seo-guidelines.md`
- `context/style-guide.md`
- `context/competitor-analysis.md`
- `context/cro-best-practices.md`

## Common Commands

- `/research [topic]`
- `/write [topic or brief]`
- `/article [topic]`
- `/rewrite [draft]`
- `/optimize [draft]`

## Typical Workflow

1. Research a topic and the SERP.
2. Turn the topic or brief into a full draft.
3. Rewrite or optimize if needed.
4. Publish only after internal links, metadata, and claims are checked.

## Example Usage

```bash
/research best curtain fabrics for summer heat
/write best curtain fabrics for summer heat
/article how to choose curtain length for living room
```

## Output Directories

- `research/` for briefs and supporting analysis
- `drafts/` for new drafts
- `rewrites/` for rewritten versions
- `published/` for finalized content

## Notes

- This repository is for TheHues consumer SEO workflows
- Write for US homeowners and home-focused shoppers unless a brief says otherwise
- Internal links should point to real public pages
