## On-Page SEO: Collection Pages

### Required Elements
1. One unique H1 aligned with primary intent.
2. Meta title with primary keyword near the front.
3. Meta description with value proposition + functional benefit.
4. Supporting intro copy that answers search intent before product grid.
5. Internal links to related guides/resources.

### Collection Intro Framework
- 80-180 words above grid (intent-focused, no fluff)
- Include primary keyword once naturally
- Include 1-2 semantic variants naturally
- Clarify user outcome (sleep, insulation, noise reduction, custom fit)

### Content Blocks to Add/Improve
- "Who this is for" block
- "How to choose" mini-guide block
- FAQ block targeting PAA-style questions
- Comparison snippets (e.g., blackout vs room darkening)

## Blog SEO Strategy (Supports Collection Rankings)

### Cluster Structure
Each of the 6 priority collections needs:
1. One collection page as commercial anchor
2. Supporting informational blog articles linked back to collection
3. Internal cross-links between related guides
4. Updated evergreen posts for freshness

### Internal Link Rules
- Include 1-2 links to target collection in relevant posts.
- Use varied, descriptive anchors.
- Add 3-6 total internal links per long-form post.
- Ensure at least one internal link appears in first third of article.

### Anchor Examples by Cluster
- Blackout: "100% blackout curtains", "blackout drapes for bedroom"
- Thermal: "thermal curtains", "insulated curtains"
- Soundproof: "soundproof curtains", "acoustic curtains"
- Custom: "custom size curtains", "tailored curtains"
- Outdoor: "outdoor curtains", "waterproof outdoor curtains"
- Sun Shade Sail: "sun shade sail", "patio shade sail"

## Keyword Usage Standards

### Keyword Placement (Blog + Collection Support Pages)
Primary keyword should appear in:
- Title/H1
- First 100 words
- At least one H2
- Conclusion or final recommendation block
- URL slug (new content)
- Meta title + meta description

### Density Guidance
- Primary keyword: natural use only; avoid forced density targets
- Secondary keywords: include where contextually relevant
- Semantic variations: prioritize readability and intent match

## Technical SEO Priorities

1. **Crawl/index hygiene**:
   - Keep canonical consistency between locale versions and primary EN pages.
   - Avoid duplicate/near-duplicate collection variants competing with primary target pages.
2. **Page speed**:
   - Optimize collection hero/media payloads.
   - Keep mobile LCP and interaction friction low.
3. **Structured data**:
   - Ensure product, breadcrumb, and organization data validity.
4. **Internal link equity**:
   - Route more contextual links to 6 priority collection URLs.

## Refresh Strategy (Quarterly)

Update cycle for priority pages and top supporting articles:
- Refresh intro copy based on evolving search intent
- Add current-year examples where relevant
- Improve FAQ coverage for new PAA patterns
- Re-check ranking anchors and internal link targets

Minimum cadence:
- Priority collection pages: monthly review
- Top support articles: every 60-90 days

## Content Quality & E-E-A-T Alignment

1. Use practical, testable advice (measurement, install, material selection).
2. Avoid unverifiable performance exaggerations.
3. Keep policy-related claims synced with current policy pages.
4. Maintain US-home context and plain-English explanations.

## SEO Checklist Per New/Updated Article

### Intent & Keyword
- [ ] Clear primary keyword and matching search intent
- [ ] Keyword naturally placed in key locations
- [ ] Secondary terms mapped to subheadings

### Structure
- [ ] One H1 and logical H2/H3 hierarchy
- [ ] Scannable formatting with lists and short paragraphs
- [ ] FAQ or concise answer sections where relevant

### Links
- [ ] 1-2 links to relevant target collections
- [ ] 2-4 additional internal links to guides/tools/pages
- [ ] External links only when they improve trust/clarity

### Conversion Support
- [ ] Includes intent-matched CTA (guide/tool/customize)
- [ ] References measurement/design support when needed

### Quality
- [ ] No fabricated data or claims
- [ ] Language aligned to TheHues brand voice
- [ ] Suitable for US English audience

Track monthly by keyword cluster:
