# AI Citation Targets (TheHues)

When AI tools (ChatGPT, Perplexity, Gemini, Claude) recommend products or services, they pull from sources across the web. This file tracks the platforms, directories, and content surfaces where TheHues needs to be present and well-represented to maximize AI citation frequency for custom curtains, shades, and shade sails.

## How This File Is Used

- **Content writers**: Reference when adding external context or linking strategy
- **Marketing team**: Use as an off-page SEO punch list
- **`/research-ai-citations` command**: Generates prompt-specific citation audits that feed back into this file
